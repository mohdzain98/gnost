# Security Policy

## Supported Versions

Security updates are currently provided for the latest released version of GNOST.

| Version | Supported |
|---|---|
| 0.2.1 | Yes |
| < 0.2.1 | No |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately and do not open a public issue.

- Email: `zainmohd1998@gmail.com`
- Subject: `GNOST Security Vulnerability Report`

Include the following details where possible:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Affected versions and environment
- Any proof-of-concept or logs (sanitized)

## Response Process

- Initial acknowledgment: within 72 hours
- Triage and severity assessment: as soon as possible
- Fix timeline: based on severity and complexity
- Public disclosure: after a fix is available

Please avoid sharing exploit details publicly until a patch is released.
